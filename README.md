# PC-Restore-Point-tutorial
A repo showing how to create a system Restore Point in case your PC's system file integrity or other features are corrupted/damaged/lost

<div align="center"> Select the "Search" icon, search "create a restore point", and select the aforementioned search</div>
 

   <p align="center">
    <img src="https://github.com/zay65/PC-Restore-Point-tutorial/blob/9528d4bcd9b245e1b522f60aa9690049ac1dc578/Search%20and%20select%20Restore%20Point.jpeg" alt="Sample Image"/>
  </p>

  <div align="center"> Under "Protection Settings", select "Local Disk (C:) (System)" and click "Configure".</div>
 

   <p align="center">
    <img src="https://github.com/zay65/PC-Restore-Point-tutorial/blob/9528d4bcd9b245e1b522f60aa9690049ac1dc578/Pre%20Sys%20RP%20Creation%20setup%20.jpeg" alt="Sample Image"/>
  </p>


<div align="center"> Select the "Turn on system protection" radio button, click "OK", and then click "Create".
</div>
 

   <p align="center">
    <img src="https://github.com/zay65/PC-Restore-Point-tutorial/blob/9528d4bcd9b245e1b522f60aa9690049ac1dc578/System%20RP%20creation%20setup.jpeg" alt="Sample Image"/>
  </p>


<div align="center"> In the ”System Protection” dialog box, in the description text box, name your Restore Point and click ”Create” to create a restore point
</div>
 

   <p align="center">
    <img src="https://github.com/zay65/PC-Restore-Point-tutorial/blob/9528d4bcd9b245e1b522f60aa9690049ac1dc578/Restore%20Point%20success.jpeg" alt="Sample Image"/>
  </p>
